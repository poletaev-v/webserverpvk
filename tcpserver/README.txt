#### ПРИМЕЧАНИЕ ####
- Программа является TCP - сервером, который принимает входящие запросы из той же локальной сети в которой запущен сервер.
- Все примеры указаны в двойных кавычках, при установке параметров двойные кавычки НЕ ИСПОЛЬЗУЮТСЯ!
- Для проверки программы нужно отправить запрос к фунционирующему серверу с данными из файла "XML"
- Для выхода из программы следует использовать комбинацию клавиш "CTRL+c"
- Скриншоты по установке находятся в архиве с программой, в папке "settings_image"

#### ПОРЯДОК РАБОТЫ ####
- Сервер в обычном состоянии создает картинку с текстом по умолчанию и сохраняет её в папке "images" (Если папка не существует, программа сама создаст папку с правами на запись и чтение)
- При поступлении запроса на сервер (принимается входящий запрос из той же сети, в которой запущен сам сервер), начинается парсинг запроса
- В принимаемом запросе сервер разбирает содержимое в виде XML и парсит все данные
- Необходимые данные текстом наносятся на картинку, которая впоследствии сохраняется в папке
- Соединение держится 10 секунд (если нет активных действий от клиента), после закрывается и вновь генерируется картинка по умолчанию

#### Установка ####
- При запуске программы "TCPserver.exe" необходимо установить 2 параметра сети
-- Первый параметр: адрес сети в которой будет обслуживаться сервер и принимать входящие запросы (По умолчанию равен "127.0.0.1" (localhost))
-- Второй параметр: адрес порта на который будут поступать запросы (По умолчанию равен "4545")
